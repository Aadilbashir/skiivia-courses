MEAN stack development

  • Category
    Full-stack development

The MEAN stack is a JavaScript-based framework for developing web applications. MEAN is named after MongoDB, Express, Angular, and Node, the four key technologies that make up the layers of the stack.

MEAN Stack consists of four main components or can say four main technologies:
M : stands for MongoDB ( Database ), mainly used for preparing document database and is a NoSQL (Non-Structured Query Language ) Database System
E : stands for Express, mainly used for developing Node.js web framework
A : stands for Angular, mainly used for developing a client-side JavaScript framework
N : stands for Nodejs, mainly used for developing the premier JavaScript web server
